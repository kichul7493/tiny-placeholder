## 프로젝트 개요

간단한 플레이스홀더 이미지가 필요할 때 사용할 수 있는 아주 작은 라이브러리.

### 기능

- 원하는 넓이와 높이를 설정할 수 있다.
- 원하는 형태(사각형, 삼각형, 원형)를 설정할 수 있다.
- 원하는 색상을 배경으로 정할 수 있다.
- 원하는 글씨를 넣을 수 있다.
